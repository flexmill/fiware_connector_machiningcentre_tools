Datei wird alle 20sec auf Maschine ausgelesen und geschrieben

Beispiel....

Spalte A:
1065_100411 = Maschine + Artikelnummer

Spalte B:
2000037 = WerkzeugID aus WinTool

Spalte C:
2 = Überwachungsart (2: Stückzahl)

Spalte D:
40 = Sollstückzahl

Spalte E:
5 = Vorwarngrenze (Maschine gibt hier eine Meldung aus)

Spalte F:
37 = Reststückzahl (zählt nach jedem Programmdurchlauf runter)
